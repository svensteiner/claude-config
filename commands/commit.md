Create a git commit for the current changes.

Steps:
1. Run `git status` to see all changes
2. Run `git diff --staged` and `git diff` to review changes
3. Run `git log --oneline -5` to see recent commit style
4. Stage relevant files with `git add`
5. Create a commit with a clear, concise message

Commit message format:
- Use imperative mood ("Add feature" not "Added feature")
- First line: max 50 chars, summarize the change
- If needed, add blank line + details

End commit message with:
```

🤖 Generated with [Claude Code](https://claude.com/claude-code)

Co-Authored-By: Claude <noreply@anthropic.com>
```

DO NOT push unless explicitly asked.
