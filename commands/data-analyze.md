Analyze data files (CSV, Excel, etc.) and generate insights.

Usage: /data-analyze <file> [focus]

Focus options: distribution, correlation, trends, anomalies

## Process
1. Load data with pandas
2. Run df.info(), df.describe()
3. Check null values and duplicates
4. Analyze based on focus area
5. Generate visualizations if helpful
6. Provide actionable insights

## For Trading Data
- Validate OHLCV structure
- Calculate technical indicators (SMA, RSI, MACD)
- Identify trends and patterns
- Check for data gaps

## Output Format
- Dataset overview (rows, columns, types)
- Key findings with numbers
- Visualizations
- Recommendations
