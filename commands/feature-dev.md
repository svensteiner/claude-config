Guided 7-phase feature development workflow.

## Phases
1. **Analyse** - Understand requirements, explore codebase
2. **Design** - Architecture, component structure
3. **Implementierung** - Write the code
4. **Test** - Add/run tests
5. **Review** - Code quality check
6. **Dokumentation** - Update docs if needed
7. **Deployment** - Prepare for release

## Agents Used
- code-explorer: Codebase analysis
- code-architect: Design decisions
- code-reviewer: Quality check

## Process
Ask user for feature description, then:
1. Explore existing code patterns
2. Design implementation approach
3. Implement incrementally
4. Test each component
5. Review for quality
6. Update documentation
7. Prepare commit

Always check existing patterns before creating new ones.
