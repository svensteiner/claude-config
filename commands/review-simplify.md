# Code Simplification Review

Finde Vereinfachungspotenzial im Code.

## Anweisungen

1. Prüfe mit `git diff` welche Dateien geändert wurden
2. Nutze den `pr-review-toolkit:code-simplifier` Agent um zu prüfen:
   - Überflüssige Komplexität
   - Duplizierter Code
   - Vereinfachbare Logik
   - Dead Code

3. Schlage konkrete Vereinfachungen vor (ohne Funktionalität zu ändern)
