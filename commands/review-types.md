# Type Design Review

Analysiere das Typen-Design der aktuellen Änderungen.

## Anweisungen

1. Prüfe mit `git diff` welche Dateien geändert wurden
2. Nutze den `pr-review-toolkit:type-design-analyzer` Agent um zu prüfen:
   - Encapsulation
   - Invariant Expression
   - Type Safety
   - Usefulness der Typen

3. Gib Ratings und Verbesserungsvorschläge für das Typen-Design
